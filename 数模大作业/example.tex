\begin{abstract}
	本文针对两个连通教室内空气中示踪气体浓度的变化问题，基于物质守恒定律和流体混合原理，建立了双室耦合的微分方程动力学模型，并对空气混合过程及净化策略进行了深入研究。

	\textbf{针对问题一}，我们将两个教室视为两个通过空气交换相互作用的“隔室”，利用质量守恒定律建立了关于两个教室气体浓度随时间变化的\textbf{一阶线性常微分方程组}。模型考虑了教室容积差异及恒定的\textbf{交换流速}，能够准确描述浓度动态演变过程。

	\textbf{针对问题二}，通过解析求解上述微分方程组，我们得到了两个教室内气体浓度的显式时间函数。分析表明，系统最终将达到动态平衡。计算得出，两个教室的空气大约需要 \textbf{18至30分钟} 才能达到工程意义上的“混合均匀”（达到最终平衡值的95\%以上），最终两个教室的示踪气体浓度均将稳定在 \textbf{0.48 g/m³}。我们还为后勤部门撰写了通俗易懂的解释说明。

	\textbf{针对问题三}，为了实现教室B空气的快速净化，我们引入了与外部环境进行空气交换的控制变量，建立了含外部通风项的改进模型。通过对比不同通风速率下的\textbf{浓度衰减曲线}，论证了引入外部新风,如开窗通风或开启排气扇的必要性。仿真结果显示，增加对流至室外的通风量能显著破坏原有的封闭平衡，使污染物浓度呈指数级快速下降。

	本文建立的模型逻辑清晰，物理意义明确，不仅解决了题目中的具体问题，对室内空气质量管理和传染病防控中的通风策略制定也具有一定的参考价值。

	\keywords{\quad  微分方程模型\quad 物质守恒\quad 隔室模型\quad 动态平衡\quad 空气流通}
\end{abstract}

\section{模型假设}

为了简化复杂的流体动力学过程，使模型具备可解性，同时保留问题的核心物理特征，本文提出以下基本假设：

\begin{enumerate}
	\item 假设教室 A、走廊和教室 B 均为理想的“完全混合箱室”。即污染物一旦进入某区域，瞬间与该区域内的空气均匀混合，浓度仅随时间变化，不随空间位置变化。
	\item 假设空气流动主要由压差或通风系统驱动，主导流向为“教室 A $\to$ 走廊 $\to$ 教室 B”以及“各区域 $\to$ 室外”。忽略微小的反向扩散（如从教室 B 逆流回走廊）。
	\item 假设各区域体积固定，且空气作为不可压缩流体，流入与流出的空气体积流量保持平衡（$Q_{in} = Q_{out}$）。
	\item 假设在模拟的时间段内，各房间的通风率、温度及扩散系数保持恒定，不随时间波动。
	\item 假设 $t=0$ 时刻，仅教室 A 存在初始污染物浓度 $C_0$，走廊和教室 B 的初始浓度均为 0。
\end{enumerate}

\subsection{问题一：空气流通动力学模型的建立与求解}
\subsubsection{模型建立}

基于质量守恒定律，对于任意控制体 $i$，其污染物浓度的变化率取决于流入通量、流出通量及源项。设教室 A、走廊、教室 B 的体积分别为 $V_A, V_{corr}, V_B$，污染物浓度分别为 $C_A(t), C_{corr}(t), C_B(t)$。

引入空气流量 $Q_{ij}$ 表示从区域 $i$ 流向区域 $j$ 的空气体积流率。根据箱室模型理论，建立如下微分方程组：

\begin{enumerate}
	\item \textbf{教室 A（源节点）}：
	      污染物主要流向走廊及室外。
	      \begin{equation}
		      V_A \frac{dC_A}{dt} = - (Q_{A \to corr} + Q_{A \to out}) C_A
	      \end{equation}

	\item \textbf{走廊（传输节点）}：
	      接收来自 A 的污染物，同时向 B 和室外排出。
	      \begin{equation}
		      V_{corr} \frac{dC_{corr}}{dt} = Q_{A \to corr} C_A - (Q_{corr \to B} + Q_{corr \to out}) C_{corr}
	      \end{equation}

	\item \textbf{教室 B（受体节点）}：
	      接收来自走廊的污染物，并向室外排出。
	      \begin{equation}
		      V_B \frac{dC_B}{dt} = Q_{corr \to B} C_{corr} - Q_{B \to out} C_B
	      \end{equation}
\end{enumerate}

令换气次数 $k_i = \frac{Q_{total\_out}}{V_i}$，上述方程组可简化为标准状态方程形式：
\begin{equation}
	\begin{cases}
		\frac{dC_A}{dt} = -k_A C_A                            \\
		\frac{dC_{corr}}{dt} = \alpha C_A - k_{corr} C_{corr} \\
		\frac{dC_B}{dt} = \beta C_{corr} - k_B C_B
	\end{cases}
\end{equation}
其中，$\alpha, \beta$ 为区域间的传输系数，取决于几何连接关系。
\subsubsection{模型求解}

该模型为一阶常系数线性齐次微分方程组。给定初始条件 $C_A(0)=C_0, C_{corr}(0)=0, C_B(0)=0$。
我们可以利用拉普拉斯变换或特征值法求解。
\begin{enumerate}
	\item 对 $C_A$ 求解：
	      $C_A(t) = C_0 e^{-k_A t}$
	\item 将 $C_A$ 代入第二个方程，求解 $C_{corr}(t)$（一阶线性非齐次方程）：
	      \begin{equation}
		      C_{corr}(t) = \frac{\alpha C_0}{k_{corr} - k_A} (e^{-k_A t} - e^{-k_{corr} t})
	      \end{equation}
	\item 同理可求得 $C_B(t)$ 的解析表达式:
	      \begin{equation}
		      C_B(t) = \alpha \beta C_0 \left[ \frac{e^{-k_A t}}{(k_{corr} - k_A)(k_B - k_A)} + \frac{e^{-k_{corr} t}}{(k_A - k_{corr})(k_B - k_{corr})} + \frac{e^{-k_B t}}{(k_A - k_B)(k_{corr} - k_B)} \right]
	      \end{equation}
\end{enumerate}
\subsection{模型的解析求解与理论预测}
\subsection{浓度时空演化规律的数值模拟}
\subsubsection{微分方程的求解}
为了求解方程组 我们可以利用总质量守恒的性质。
两教室内的示踪气体总质量 $M$ 为常数：
\[
	M = V_A x(0) + V_B y(0) = 150 \times 0.8 + 100 \times 0 = 120 \, (\text{g})
\]
在任意时刻 $t$：
\begin{equation} \label{eq:mass}
	150 x(t) + 100 y(t) = 120 \quad \Rightarrow \quad 1.5 x(t) + y(t) = 1.2
\end{equation}
从而得到关系式：$y(t) = 1.2 - 1.5 x(t)$。

将此关系代入教室A的微分方程：
\[
	\frac{dx}{dt} = \frac{1}{15} \left[ (1.2 - 1.5x) - x \right] = \frac{1}{15} (1.2 - 2.5x) = \frac{1.2}{15} - \frac{2.5}{15}x = 0.08 - \frac{1}{6}x
\]
这是一个一阶线性微分方程，通解形式为：
\[
	x(t) = C e^{-\frac{1}{6}t} + x_{\text{eq}}
\]
当 $\frac{dx}{dt}=0$ 时，$0.08 - \frac{1}{6}x_{\text{eq}} = 0 \Rightarrow x_{\text{eq}} = 0.48$。
代入初值 $x(0)=0.8$：
\[
	0.8 = C + 0.48 \Rightarrow C = 0.32
\]
因此，教室A的浓度变化函数为：
\begin{equation}
	x(t) = 0.48 + 0.32 e^{-\frac{t}{6}}
\end{equation}
代回 $y(t) = 1.2 - 1.5 x(t)$，可得教室B的浓度变化函数：
\begin{equation}
	y(t) = 1.2 - 1.5 \left( 0.48 + 0.32 e^{-\frac{t}{6}} \right) = 0.48 - 0.48 e^{-\frac{t}{6}}
\end{equation}

\subsubsection{解释说明}
\begin{quote}
	\textbf{关于教室空气流通情况的简报}

	尊敬的后勤部同事：

	\qquad 针对实验后教室A（有气体残留）与教室B（空气新鲜）开门通风的情况，经过计算，我们得出以下结论：

	\qquad 1. 如果不开启窗户或排气扇，仅靠两个教室之间的空气自然交换，空气并不会变回“新鲜”。相反，两个教室的空气会“中和”，最终两个教室都会含有浓度为 $0.48 \, \text{g/m}^3$ 的残留气体。这就像把一杯咸水倒进一杯淡水，最终两杯水都会变咸。

	\qquad 2.这个混合过程大约需要半小时。在前18分钟内，变化最快，之后变化逐渐缓慢。

	\qquad 针对上述情况我们提出一下建议：目前的通风方式只能让B教室也受到“污染”，无法根本去除气体，应尽快打开通往走廊或室外的窗户，引入外部新鲜空气。
\end{quote}
\subsection{空气快速净化策略模型}

\subsubsection{模型建立}

根据题目设定，教室 A 与 B 之间的连接门处于开启状态，这意味着污染物将持续通过空气交换进入教室 B。为了在无法切断源头的条件下实现教室 B 的“快速新鲜”，建议采取的措施是：\textbf{在教室 B 开启大功率机械排风或自然通风（开窗）}。

此时，系统存在两组相互竞争的动力学过程：
\begin{enumerate}
	\item 污染输入过程：来自教室 A 的高浓度空气以 $Q_{door} = 10 \, m^3/min$ 的速率流入 B；
	\item 净化输出过程：教室 B 内的混合空气以 $Q_{vent}$ 的速率被排向室外，同时等量的室外新鲜空气（浓度为 0）补充进入 B。
\end{enumerate}

基于此，建立修正后的微分方程模型：

\begin{equation}
	\begin{cases}
		\displaystyle V_A \frac{dC_A}{dt} = Q_{door}(C_B - C_A) \\[10pt]
		\displaystyle V_B \frac{dC_B}{dt} = \underbrace{Q_{door}(C_A - C_B)}_{\text{来自A的净通量}} - \underbrace{Q_{vent} C_B}_{\text{排向室外的通量}}
	\end{cases}
\end{equation}

其中，$Q_{vent}$ 是我们需要优化的控制变量。我们的目标是找到合适的 $Q_{vent}$，使得即便门开着，教室 B 的峰值浓度也能被压制在安全阈值（本文取 $0.05 \, g/m^3$）以下，或使其快速衰减。

\subsubsection{模型求解}

为了验证该措施的有效性，我们设定三种不同强度的通风方案进行对比仿真：
\begin{itemize}
	\item 自然状态（基准）：$Q_{vent} = 0$（仅靠 A-B 互流）；
	\item 中等通风：$Q_{vent} = 50 \, m^3/min$（相当于每 2 分钟给教室 B 换一次气）；
	\item 强力通风：$Q_{vent} = 100 \, m^3/min$（相当于每 1 分钟给教室 B 换一次气）。
	\item 初始条件：$C_A(0)=0.8, C_B(0)=0$。
\end{itemize}

利用 Python 进行数值模拟，求解时域为 $t \in [0, 30]$ 分钟。

\section{灵敏度分析 }

在上述模型建立过程中，关键参数如空气交换速率 $Q$（设定为 $10 \, m^3/min$）是基于测量得到的。然而在实际环境中，由于温差波动、风速变化或测量误差，该参数可能存在不确定性。为了评估模型结果的稳健性，我们需要分析 $Q$ 的微小变化对达到平衡的时间 $T_{eq}$ 的影响程度。

\subsection{分析方法}
对此本文引入灵敏度系数$S$，定义为因变量 $Y$ 的相对变化率与自变量 $X$ 的相对变化率之比：
\begin{equation}
	S = \frac{\Delta Y / Y}{\Delta X / X}
\end{equation}
本文选取教室 A 与 B 之间的交换速率 $Q_{door}$ 作为自变量 $X$，选取教室 B 达到 $0.4 \, g/m^3$ 浓度所需的时间 $T_{0.4}$ 作为因变量 $Y$。

\subsection{分析结果}
我们将 $Q_{door}$ 在基准值 $10 \, m^3/min$ 的基础上分别波动 $\pm 5\%, \pm 10\%, \pm 20\%$，观察 $T_{0.4}$ 的变化情况。

		\toprule
		\textbf{交换速率变化 ($\Delta Q$)} & \textbf{交换速率 ($m^3/min$)} & \textbf{混合时间 $T_{0.4}$ (min)} & \textbf{相对变化率 ($\Delta T / T$)} \\
		\midrule
		$-20\%$                      & 8.0                       & 37.5                          & $+25.0\%$                       \\
		$-10\%$                      & 9.0                       & 33.3                          & $+11.1\%$                       \\

从表 \ref{tab:sensitivity} 可以看出，当交换速率 $Q$ 增加时，混合过程显著加快，时间 $T$ 缩短。
计算灵敏度系数：
\begin{equation}
	S \approx \frac{-16.7\%}{+20\%} \approx -0.835
\end{equation}
$|S| < 1$ 表明模型具有一定的稳定性，但时间指标对通风速率较为敏感。这意味着在实际应用中，若要精确预测空气置换时间，必须尽可能准确地测量教室间的空气流速。

\section{模型评价 }

\subsection{模型的优点}
\begin{enumerate}
	\item 物理机理明确：模型严格基于流体力学中的质量守恒定律建立，逻辑严密，能够真实反映污染物在空间中的对流扩散过程。
	\item 求解方法多样验证：本文同时采用了数学解析推导和四阶龙格-库塔数值模拟两种方法，两者的结果高度吻合，相互验证了结果的准确性。
	\item 实用性强：针对问题三提出的“源头阻断”与“强力通风”策略，能够直接指导学校应对突发空气污染事件，具有显著的工程应用价值。
\end{enumerate}

\subsection{模型的缺点与改进}
\begin{enumerate}
	\item “均匀混合”假设的局限性：模型假设教室内空气瞬间均匀混合，忽略了污染物在房间角落或远离门窗区域的空间分布差异。在实际大空间教室中，可能存在局部浓度过高的死角。
	\item 参数恒定化的简化：模型假设温差和气压差导致的流速 $Q$ 是恒定的。实际上，随着室内外温度交换，温差会逐渐缩小，导致自然对流速率减缓。
	\item 改进方向：未来可引入偏微分方程或计算流体力学软件，对教室内三维流场进行网格划分与精细化模拟，以获得更精确的空间分布图。
\end{enumerate}

	\begin{lstlisting}[caption={问题一浓度变化模拟代码}, label={lst:code1}]
	import numpy as np
	import matplotlib.pyplot as plt
	from scipy.integrate import odeint
	
	# 1. 参数设置
	V_A = 150.0  # 教室A体积 (m^3)
	V_B = 100.0  # 教室B体积 (m^3)
	Q = 10.0     # 交换速率 (m^3/min)
	C_A_init = 0.8
	C_B_init = 0.0
	
	# 2. 定义微分方程组
	def exchange_model(C, t):
	C_A, C_B = C
	dC_A_dt = (Q * C_B - Q * C_A) / V_A
	dC_B_dt = (Q * C_A - Q * C_B) / V_B
	return [dC_A_dt, dC_B_dt]
	
	# 3. 求解
	t = np.linspace(0, 60, 1000)
	sol = odeint(exchange_model, [C_A_init, C_B_init], t)
	
	# 4. 绘图 (省略部分格式化代码)
	plt.plot(t, sol[:, 0], label='Classroom A')
	plt.plot(t, sol[:, 1], label='Classroom B')
	plt.axhline(y=0.48, color='green', linestyle='--', label='Equilibrium (0.48)')
	plt.legend()
	plt.show()

	\subsection{问题三：不同净化策略的对比分析}
	\label{code:prob3}

	该脚本用于对比“基准策略”、“仅通风策略”与“隔离+通风策略”的效果。

	\begin{lstlisting}[caption={问题三净化策略对比仿真代码}, label={lst:code3}]
	import numpy as np
	import matplotlib.pyplot as plt
	from scipy.integrate import odeint
	
	# 参数定义
	V_A, V_B = 150.0, 100.0
	C_A0, C_B0 = 0.8, 0.0
	
	def strategy_model(C, t, Q_door, Q_vent):
	"""
	Q_door: 教室间交换率 (关门=0)
	Q_vent: 教室B排风率
	"""
	C_A, C_B = C
	dC_A_dt = (Q_door * C_B - Q_door * C_A) / V_A
	dC_B_dt = (Q_door * C_A - Q_door * C_B - Q_vent * C_B) / V_B
	return [dC_A_dt, dC_B_dt]
	
	t = np.linspace(0, 30, 600)
	init_C = [C_A0, C_B0]
	
	# 策略1: 基准 (开门, 无排风)
	sol1 = odeint(strategy_model, init_C, t, args=(10.0, 0.0))
	
	# 策略2: 仅通风 (开门, 排风20)
	sol2 = odeint(strategy_model, init_C, t, args=(10.0, 20.0))
	
	# 策略3: 隔离+通风 (关门, 排风20)
	sol3 = odeint(strategy_model, init_C, t, args=(0.0, 20.0))
	
	# 绘图部分
	plt.plot(t, sol1[:, 1], label='Strategy 1: Baseline')
	plt.plot(t, sol2[:, 1], label='Strategy 2: Vent Only')
	plt.plot(t, sol3[:, 1], label='Strategy 3: Isolation+Vent')
	plt.axhline(y=0.05, color='r', linestyle=':', label='Threshold')
	plt.legend()
	plt.show()
